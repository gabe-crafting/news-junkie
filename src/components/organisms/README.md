# Organisms

This folder contains Storybook stories for **organism-level components**.

In atomic design, organisms are complex UI components that combine molecules and atoms to form distinct sections of an interface.

## Examples of Organisms

- Navigation bars
- Search forms
- Product cards
- Comment sections
- Data tables
- Form sections

## Adding Stories

Create your organism component stories in this folder following the naming convention:
- `ComponentName.stories.tsx` or `ComponentName.stories.ts`

Stories will automatically be picked up by Storybook based on the configuration in `.storybook/main.ts`.
